# DiscordBot
 A basic discord bot.
 Runs predefined commands.
 
## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to run files.

```bash
pip install discord
pip install python-dotenv
```

## Usage

```python
from dotenv import load_dotenv
from discord import Intents, Client, Message

# return
load_dotenv()

```

# Execution
```python
python main.py
```
